技能描述

使用声骸技能，召唤「角」腾跃而起，对目标造成48.64% 衍射伤害并降下光雷，光雷对周围目标造成最多五段，每段19.46% 衍射伤害。

随后「角」向下盘旋攻击周围敌人，连续两次造成48.64% 衍射伤害。

施放声骸技能时，获得「光阴之祝」，持续15秒。

「光阴之祝」持续期间： 自身的共鸣技能伤害加成提升16.00%；

 自身的共鸣技能命中目标后，目标每秒受到1次16.00% 衍射伤害，此次伤害为角色的共鸣技能伤害，持续15秒。

冷却时间：20秒
